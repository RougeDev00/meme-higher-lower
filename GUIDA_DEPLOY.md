# Guida alla Pubblicazione del Sito (Deployment)

Questa guida ti spiega passo dopo passo come mettere online il tuo gioco "Meme Higher Lower" collegandolo a un dominio personalizzato (es. `tuogioco.com`) e configurando il database per i punteggi.

## 1. Preparazione del Database (Supabase)

Il gioco usa Supabase per salvare la classifica. Se non l'hai già fatto:

1.  Vai su [database.new](https://database.new) e crea un nuovo progetto gratuito.
2.  Una volta creato, vai su **SQL Editor** (icona dalla barra laterale sinistra).
3.  Copia e incolla questo codice SQL e clicca **Run**:

```sql
create table if not exists leaderboard (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  wallet_address text unique not null,
  username text not null,
  score integer default 0
);

alter table leaderboard disable row level security;
```

4.  **Region**: Se la maggior parte dei tuoi utenti è in America, seleziona **US East (N. Virginia)**. Se sono in Europa, scegli **Central EU (Frankfurt)**.
5.  Vai su **Project Settings** (ingranaggio) -> **API**.
6.  Tieniti pronti i valori di `Project URL` e `anon public key`. Ti serviranno tra poco.

## 2. Pubblicazione su Vercel

Vercel è il servizio migliore e gratuito per ospitare siti Next.js come il tuo.

1.  Crea un account su [Vercel.com](https://vercel.com) (puoi usare GitHub).
2.  Assicurati che il tuo codice sia su GitHub. (Se stai lavorando in locale, carica il progetto su una repository GitHub).
3.  Nella dashboard di Vercel, clicca **Add New** -> **Project**.
4.  Seleziona la tua repository `meme-higher-lower`.
5.  Configura il progetto:
    *   **Framework Preset**: Next.js (dovrebbe rilevarlo in automatico).
    *   **Environment Variables**: Qui devi inserire le chiavi di Supabase.

    Aggiungi due variabili:
    *   Nome: `NEXT_PUBLIC_SUPABASE_URL` | Valore: `(Il tuo Project URL copiato prima)`
    *   Nome: `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Valore: `(La tua anon public key copiata prima)`

6.  Clicca **Deploy**.
7.  Attendi un minuto. Se tutto va bene, vedrai i fuochi d'artificio e il tuo sito sarà online su un dominio tipo `meme-higher-lower.vercel.app`.

## 3. Collegare un Dominio Personalizzato

Ora che il sito è online, puoi collegare un dominio vero (es. `.com` o `.it`).

1.  **Compra il dominio**: Vai su un sito come Namecheap, GoDaddy, o Aruba e compra il dominio che ti piace.
2.  Torna su **Vercel** -> Vai al tuo progetto -> **Settings** -> **Domains**.
3.  Inserisci il tuo dominio (es. `ilmiogioco.com`) e clicca **Add**.
4.  Vercel ti darà dei parametri DNS da configurare. Di solito sono:
    *   **Tipo**: A Record
    *   **Nome**: @
    *   **Valore**: 76.76.21.21
    *   *(Oppure CNAME se usi un sottodominio www)*

5.  Vai sul sito dove hai comprato il dominio (es. Namecheap), cerca "DNS Management" o "Gestione DNS".
6.  Aggiungi i record che ti ha dato Vercel.
7.  Dopo qualche minuto (o ora), il tuo dominio funzionerà e punterà al tuo gioco!

## Riepilogo Funzionamento

*   **Utente**: Entra, scrive Username e incolla il suo Wallet.
*   **Gioco**: Salva questi dati. Quando fa punti, li invia a Supabase.
*   **Leaderboard**: Legge da Supabase e mostra i migliori.
*   **Sito**: È ospitato sui server veloci di Vercel.

Se hai problemi con uno di questi passaggi, chiedimi pure!
